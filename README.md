# 🎯 QAGlobalHunt — AI-Powered SQA Career Platform

Find remote & relocation SQA jobs worldwide using AI.

## Features
- 📄 PDF Resume Upload → ATS Optimization  
- ✍️ Manual Skills Entry  
- 🌍 AI Job Hunting (LinkedIn, Himalayas, Remote.co, Relocate.me etc.)  
- ✈️ Remote + Visa-sponsored Relocation filter  
- 🚫 Israel excluded automatically  

## Setup (1 minute)

### Option 1: Vercel (Easiest)
1. Push to GitHub
2. vercel.com → Import repo → Deploy
3. Done ✓

### Option 2: Netlify  
1. Drag `dist/` folder to netlify.com/drop
2. Done in 30 seconds ✓
